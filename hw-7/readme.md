# Написание плагина для Gradle

## Инструкция:
1. Создать ветку homework-7, переключиться на нее
2. Создайте модуль hw07-gradle
3. Создать конфигурацию для плагина через extension, передавать входные параметры: значение поля в классе, название класса. Также в параметрах передавать рутовую директорию для кодогенерации и название пакета для сгенерированного класса. Директорию, в которой осуществляется кодогенерация, добавим в основной sourceSet.
4. \* Генерировать класс в generated sources используя gradle api
5. Создать саму Gradle-таску, которая будет выполнять кодогенерацию. 
5. Регистрируем кодген-таску перед compile
6. Написать интеграционный тест для созданного плагина
6. Написать функциональный тест для созданного плагина

*Формат сдачи: *
- ссылка на PR в github

## Критерии оценивания
1. Задание сдано на проверку - 1 балл
2. Создан PR - 1 балл
3. Реализован gradle-плагин для кодогенерации -4 балл
4. Реализованы интеграционные тесты - 2 балл
5. Реализованы функциональные тесты - 2 балл
6. Реализовано дополнительное задание со \* - 2 балла
